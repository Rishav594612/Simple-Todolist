# Todo List App
A simple, beginner-friendly Todo List web application built using **HTML**, **CSS**, and **JavaScript**.  
Tasks are saved in the browser using **localStorage**, so they stay even after refreshing the page.

## Features
- Add new tasks  
- Mark tasks as completed or active  
- Edit tasks  
- Delete tasks  
- Filter: **All**, **Active**, **Completed**  
- Clear completed tasks or clear all tasks  
- Persistent storage with `localStorage`

## File Structure
project-folder/
│── index.html
│── style.css
└── script.js

## How to Run
1. Download all three files into the same folder.  
2. Open **index.html** in any web browser.  
3. Start adding and managing your tasks.

## Technologies Used
- HTML5  
- CSS3  
- Vanilla JavaScript
